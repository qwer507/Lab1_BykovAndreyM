Проверка данных пользователя при регистрации
Реализация валидации пользовательских данных при регистрации на C# с комплексным логированием.

Возможности
Валидация логина в одном из трёх форматов:

Номер телефона (+X-XXX-XXX-XXXX)

Адрес электронной почты (xxx@xxx.xxx)

Обычная строка (латиница, цифры и знак подчёркивания, мин. 5 символов)

Валидация пароля со строгими требованиями:

Минимум 7 символов

Только кириллица, цифры и спецсимволы

Обязательно: одна буква в верхнем регистре, одна в нижнем, одна цифра, один спецсимвол

Подтверждение пароля (совпадение с паролем)

Предустановленный список запрещённых логинов

Детальные сообщения об ошибках (10+ различных причин)

Двойное логирование (консоль + файл) с маскированием паролей

Требования к входным данным
Поле	Требования
Логин	Телефон: формат +X-XXX-XXX-XXXX
Email: формат xxx@xxx.xxx
Строка: мин. 5 символов, только a-z, A-Z, 0-9, _
Не должен совпадать с логинами из предустановленного списка
Пароль	Мин. 7 символов
Допустимые символы: кириллица (а-я, А-Я), цифры (0-9), спецсимволы
Обязательно: буква в верхнем регистре, буква в нижнем регистре, цифра, спецсимвол
Подтверждение пароля	Должно полностью совпадать с паролем
Выходные данные
Выход	Описание
Строка 1	"True" — при успешной регистрации, "False" — при ошибке
Строка 2	Пустая строка — при успехе; текст ошибки с пояснением причины — при неудаче
Логирование
При успешной регистрации: дата-время запроса, логин, маскированный пароль, маскированное подтверждение, фраза «Успешная регистрация»

При неуспешной регистрации: дата-время запроса, логин, маскированный пароль, маскированное подтверждение, текст ошибки (включая трассировку стека исключения, если есть)

Куда пишем: в файл и в консоль

Маскирование паролей: детерминированное — одинаковые пароли дают одинаковый результат маскирования, разные пароли — разный

Возможные причины ошибок (10+)
Неверный формат телефона

Неверный формат email

Неверный формат строки (недопустимые символы)

Логин слишком короткий (менее 5 символов)

Логин содержит недопустимые символы

Логин находится в запрещённом списке

Пароль слишком короткий (менее 7 символов)

В пароле отсутствует буква в верхнем регистре

В пароле отсутствует буква в нижнем регистре

В пароле отсутствует цифра

В пароле отсутствует спецсимвол

Пароль содержит недопустимые символы

Пароль и подтверждение пароля не совпадают

Технологический стек
Язык: C# (.NET)

Среда разработки: Visual Studio / Visual Studio Code

Логирование: Serilog (или стандартные методы вывода в консоль и записи в файл)

Система контроля версий: GitHub
